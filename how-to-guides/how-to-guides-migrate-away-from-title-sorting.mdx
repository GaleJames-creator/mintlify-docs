---
title: 'Migrate away from title sorting'
description: 'Update your code to stop using title sorting in BookHub Publisher API v2. Use date sorting or client-side sorting instead.'
---

## Step 1: Audit your codebase for `sort=title` parameters

```bash
# Search your codebase
grep -r "sort=title" .
grep -r "sort.*title" .
```

## Step 2: Replace with date sorting or client-side sorting

```python
# Before
response = get_books(sort='title', order='asc')

# After - Option A: Use date sorting
response = get_books(sort='createdDate', order='desc')

# After - Option B: Client-side sort
all_books = get_books()
sorted_books = sorted(all_books, key=lambda x: x['title'])
```

## Step 3: Update documentation and inform your team

* Update any internal documentation that references title sorting.
* Communicate the change to your team to ensure everyone is aware of the new sorting approach.
