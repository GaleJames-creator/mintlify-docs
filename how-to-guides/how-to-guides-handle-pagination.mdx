Retrieve paginated results when fetching all books from the BookHub API. 

## Prerequisites

<Prerequisites />

* Familiarity with BookHub Publisher API v2.
* A valid JWT for authentication.
* Access to your codebase that integrates with the BookHub Publisher API v2.

## Step 1: Send a paginated request

For an overview of how pagination works in the BookHub API, see [Pagination](/conceptual-guides/conceptual-guides-pagination).

The examples use `page` and `limit` only. The endpoint also accepts `status`, `sort`, and `order` parameters — see [API endpoints](/reference/reference-api-endpoints#get-all-books) for details.

```python
def fetch_all_books(base_url, token):
    all_books = []
    page = 1
    while True:
        response = requests.get(
            f"{base_url}/v2/books",
            params={'page': page, 'limit': 100},
            headers={'Authorization': f'Bearer {token}'}
        )
        response.raise_for_status()
        data = response.json().get('data', {})
        if 'pagination' not in data:
            break
        all_books.extend(data.get('books', []))
        if page >= data['pagination']['totalPages']:
            break
        page += 1
    return all_books
```

<Note>The loop exits when `page` reaches `totalPages`, including when the API returns fewer items than `limit`. If the response does not include a `pagination` object, the loop exits immediately.</Note>

### Error responses

The `page` parameter defaults to 1. The `limit` parameter accepts values between 1 and 100 and defaults to 20.

| Status code | Meaning |
|-------------|---------|
| `400` | Bad request — invalid `page` or `limit` parameter |
| `401` | Unauthorized — missing or invalid JWT token |
| `429` | Rate limit exceeded — 150 requests per minute |

## Step 2: Test your implementation

Call the `fetch_all_books` function with your API base URL and JWT. Verify that it retrieves all books across multiple pages by checking the length of the returned list against the `totalItems` field in the pagination object.

```python
BASE_URL = "https://api.bookhub.com/api"
TOKEN = "YOUR_JWT_TOKEN"
books = fetch_all_books(BASE_URL, TOKEN)
print(f"Total books fetched: {len(books)}")
```
