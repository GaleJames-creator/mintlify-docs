Authenticate with the BookHub Publisher API using JWT bearer tokens.

## JWT bearer token

All API requests require authentication using a JWT bearer token.

### Obtaining a token

Email api-support@bookhub.com with your publisher account information.

### Using the token

Replace `YOUR_JWT_TOKEN` with the 24-hour JWT provided by api-support@bookhub.com.

```http
Authorization: Bearer YOUR_JWT_TOKEN
```

```bash
curl -X GET https://api.bookhub.com/api/v2/books \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### Token expiration

* Tokens expire in 24 hours and require renewal.
* A `401 Unauthorized` response indicates an expired token.

### Token renewal

Tokens expire after 24 hours. To renew, email api-support@bookhub.com with your publisher account information to request a new token before your current one expires.

### Token security

Store tokens securely and never expose them in client-side code, version control, or logs. Rotate tokens immediately if compromised by contacting api-support@bookhub.com.
