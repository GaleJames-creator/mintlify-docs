This section explains how to authenticate with the BookHub Publisher API using JWT bearer tokens. 

## JWT bearer token

All API requests require authentication using a JWT bearer token.

### Obtaining a token

Obtain your JWT credentials by contacting:

* Send an email to api-support@bookhub.com
* Include your publisher account information.

### Using the token

Include the token in the `Authorization` header:

```http
Authorization: Bearer YOUR_JWT_TOKEN
```

### Authenticate a request

```bash
curl -X GET https://api.bookhub.com/api/v2/books \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

### Token expiration

* Tokens expire after 24 hours, so they must be refreshed regularly.
* A `401 Unauthorized` response indicates an expired token.
